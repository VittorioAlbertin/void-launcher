# VoidLauncher Releases

This folder contains the APK files for VoidLauncher.

## Installation

1. Download the latest APK file from this folder
2. Transfer it to your Android device
3. Install the APK (you may need to enable "Install from unknown sources")
4. Follow the setup instructions in [QUICKSTART.md](../QUICKSTART.md)
