# VoidLauncher Releases

This folder contains the APK files for VoidLauncher.

## Installation

1. Download the latest APK file from this folder
2. Transfer it to your Android device
3. Install the APK (you may need to enable "Install from unknown sources")
4. Follow the setup instructions in [QUICKSTART.md](../QUICKSTART.md)

## Version History

### v0.3 (Latest) - 2025-11-11
**File:** `VoidLauncher-v0.3.apk`

**Changes:**
- **Biometric Unlock Support**: Double-tap lock now uses Accessibility Service instead of Device Admin, allowing biometric (fingerprint/face) unlock after locking
- **New Default Gestures**:
  - Swipe up → All Apps menu (previously Chrome)
  - Swipe right → Camera (previously Messages)
- **Double Tap Lock Default**: Now enabled by default for new installations

**Setup Notes:**
- After installation, enable the Void Launcher Accessibility Service in Settings → Accessibility → Void Launcher to use double-tap lock with biometric unlock
- Existing users will need to clear app data or reinstall to get new defaults

### v0.2 - 2025-11-11
**File:** `VoidLauncher-v0.2.apk`

Initial release with core features
